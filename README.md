# 📝 Todo System (MERN Stack)

A simple Todo application built using the MERN stack.  
This project was created while I was learning full-stack development and is now being shared with a clean GitHub repository.

## 🚀 Features
- Add new todos
- Edit Todo
- Delete todos
- Basic CRUD operations using REST API
- MongoDB database integration
- Mongoose models

## 🛠️ Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
